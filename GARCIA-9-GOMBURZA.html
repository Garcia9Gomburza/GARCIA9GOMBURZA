Website Content

  • Lists

    -In HTML, there are two main types of lists: unordered lists - the list items are marked with bullets ordered lists - the list items are marked with numbers or letters The CSS list properties allow you to: Set different list item markers for ordered lists Set different list item markers for unordered lists Set an image as the list item marker Add background colors to lists and list items.

  • Links

    -HTML Links - Hyperlinks HTML links are hyperlinks. You can click on a link and jump to another document. When you move the mouse over a link, the mouse arrow will turn into a little hand.The HTML style attribute is used to add styles to an element, such as color, font, size, and more.
